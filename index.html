Week 1

Introduction to class, Project 1

Homework

  • Come up with a natural system you'd like to represent in code: sun moving across the sky, a bunch of ants on a table, etc. The system need to have several moving components. Think about how you could use a keyboard and/or a mouse to make your system interactive.
  • Create a diagram of the system that would show the following: All components of the system (including the user interacting with it); The roles of each component - what do they do? The relationships of the components - how do they affect each other?
  • Here is an example of the system of ants on a table
