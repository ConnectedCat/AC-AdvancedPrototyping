Project briefs for Advanced Interactive Prototyping

Project 1 - Representation

Programming is a form of self-expression, a way to represent the world you see through your eyes - like painting, poetry, or architecture. And just like those endeavors it comes with its own set of requirements, opportunities, and limitations. The goal of this project is to explore these opportunities and limitations by comparing programming with over forms of self-expression. We'll create representations of a natural phenomenon in different media, compare them, and find was to engage our imagination in representing it. It could take the form of a "what if" scenario, a metaphor, a re-imagination or juxtaposition - yours is the poetic license.

Tools and materials we'll use

Processing IDE, Miro diagramming software

Principles and concepts you'll learn

Object-oriented programming, UML and flowchart diagrams, simulation and representation in code

Project 2 - Control

The underlying purpose of any interface is control. User's wishes mapped onto the system's capabilities. This project focuses on the structure of this mapping, from the micro-scale (a single point of interaction), to medium - feature list, layout, to macro - modularity and customization. We'll explor micro-interactions, contextual controls, gestalt principles in interface design and the underlying system capabilities that could afford a set of interface options.

Tools and materials we'll use

TinkerCAD, Processing IDE, ESP32 microcontroller, sensors, Arduino IDE, Serial Monitor

Principles and concepts you'll learn

Serial communication, data packets, communication and control

Project 3 - Communication

The ability to communicate is at the very core of what makes us human - and we imparted this skill on our machines as well. The Internet of Things - as overused as this term is - means that ANY machine with electronic circuitry would be able to communicate. The Internet is the language and data is its grammar. The role of a designer is akin to that of an interpreter - making it easier for humans to talk to machines. In this project we will work with collecting and transmitting data, as well as representing it in the form of a dashboard.

Tools and materials we'll use

JavaScript/P5JS, ESP32 microcontroller, sensors, Adafruit IO cloud service, IfTTT cloud service

Principles and concepts you'll learn

Computing cloud, Internet of Things, APIs, data presentation, event triggers

Project 4 - Exploration

This is an opportunity for you to dig deeper in to any of the previous 3 topics - or create a high-fidelity prototype of any other of your interactive projects. The scope of this project includes researching the precedents and the related technologies, imagining and scetching out a use-case scenario, creating a diagam schematic and buildig a prototype to your own specifications.
